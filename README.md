# Causal Deep Learning: Code Examples

This repository contains the Python code examples that accompany the book "Causal Inference for Machine Learning Engineers - A Practical Guide" by Durai Rajamanickam.

## Organization

The code is organized by chapter. Each chapter directory contains one or more Python files that demonstrate the key concepts discussed in that chapter.

## Requirements

* Python 3.x
* Libraries: numpy, pandas, matplotlib, statsmodels, torch, scikit-learn, networkx (for Chapter 4, if included)

You can install the necessary libraries using pip:

```bash
pip install numpy pandas matplotlib statsmodels torch scikit-learn networkx
